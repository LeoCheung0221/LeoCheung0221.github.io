For FUTURE

发表于 2016-11-27 |

前言

装修一直是互联网极力想颠覆的领域之一，十年之前，大量的建材采购电商就已兴起。然而直至今日，互联网家装仍难以见到大规模的发展趋势。其实家装就是房地产下游产业，要想解决用户需求，首先要了解的就是互联网家装发达地区，没错，就是北上广深杭这些地方的买房人群。

以上是文章摘要 以下是余下全文

自2016年“去库存”政策一出，房地产行业整体呈现回暖趋势，随之一起的家装行业也迎来春天。与传统家装思维不同的是，生活在这些倍感压力的城市，买房本属不易，却不想装修又费神费力，因此势必会顺应O2O的懒人经济潮流，这直接导致他们非常重视消费体验。
这样的情况下，未来的国内家装O2O呈现一种趋势－一站式整体家装，从建材到设计，从装修到家居，这里面行业分散带来的弊病才是用户需求的痛点所在，家居建材行业无法直接对接到C端用户，而作为家装服务流程中的一环，打通家装产业链，对接上下游资源，自然就应运出了一站式整体家装模式。
这里稍加解释，所谓的家装业上下游产业即上游的家具销售、家居品销售、家电销售，下游的家装设计服务。

定位

互联网消费本就是中低端消费，切入点围绕预算，视工头为核心
家装定位: 追求高端 – 实践中端 – 兼顾低端

私人定制型模式：有钱人（毕竟少数）
传统家装型模式：固定家庭（中坚力量）
现代快装型模式：不固定家庭（崛起势力）
最好的发展：重点发展现代快装型模式家装，择优选择传统家装型模式家装，适当选择发展私人定制型模式家装

传统套路

六步法则

1)  寻找家装照片，从视觉入手
2） 上网搜索，寻找关于如何装修自己理想效果有关的信息
3） 搜索结果信息量大而广，无从入手
4） 思考自己是否可以负担费用
5） 如果能负担，最好的解决是通过身边的朋友，找到好承包商
6） 承包商报价，得知预算

不断摸索，连接消费者和行业人士
个人更倾向 消费者连接消费者（UGC）引导 ，建立社群，实现运营，自主导向流量
前期用户获取，考虑当地房产中介，这个获取成本很低而且迅速精准

头脑风暴

解决传统模式下出现的各个痛点

1. 估价不准
   估价精准 -> 费用负担
   估价不准 -> 平台信誉 -> 基于大数据分析

2. 费用负担
   负担得起 -> 寻找承包商 -> 承包商信誉
   负担不起 -> 是否贷款 -> 商家/银行 优惠

3. 有无优惠
   含有优惠 -> 承包商优惠/银行政策优惠 
    * 承包商方面（见附录）
      分期、大礼包、满多少送多少 -> 自主制定，吸引客户
    * 银行方面（见附录）
      各大行装修贷 -> 适时引入宣传，降低客户周转

   没有优惠 -> 几乎很少能打动用户，除非美在价格上

4. 承包商信誉
   口口相传，是提升用户留存率之根本，甚至决定企业生死，所以做好如下事情：
   1）技术专业，提供资质证明
   2）主动与客户交流，满足客户
   3）沟通好当地房地产经纪人
   4）公司官网展示案例

5. 售后服务
   1）签订履行服务合同
   2）做好客户回访工作

用户real需求:

国内主流推广渠道

  • 公司官网
    购买域名建站
    优点：公司门户，业务案例展示，提高企业知名度、信誉度
    缺点：前期访问流量少，

  • 淘宝 & 天猫（天猫家装）
    淘宝开店铺： 流程参考文章
    天猫开旗舰店： 流程参考文章
    优点: 入口流量大，开店成本运营较低，方便推广
    缺点: 线上竞争激烈，受制于信誉评级

  • 齐家 & 土巴兔 & 一起装修网
    入驻商家平台
    优点：专业家装线上平台，客源多，便于推广运营
    缺点：同行恶意竞争打压，业务不稳定，同样受制于信誉评级

  • 微信
    公众号: 流程参考文章
    服务号: 流程参考文章
    应用号(MiniApp): 流程参考文章
    优点：潜在客户群庞大，易于客户维护，适合群体销售
    缺点：前期流量引导较难，短时间内提升信誉度较难

  • 类似知乎、房天下等大型问答社区
    运营推广
    优点：耗费极少人力，宣传成本低，点对点解决用户需求
    缺点：引导客户难，成单率低，时效性低

商业模式

让客户写下需求 商家去主动满足的模式
正确的时间引入正确的行业
花钱的过程越简单越好

About Future

线上运营(推荐组合):

官网 + 淘宝店铺/天猫旗舰店 + 微信公众号/服务号 + 朋友圈创意H5营销

线下部署:

调查闵行区大小装修公司的关于材料费/人工费/设计费/质检费的统计，对比行业水平，制定合适价格 -> 预备长期价格战
应用大数据分析主流装修风格，装修群体 -> 互联网+
设计资源专利保护 -> 创新
打通行业相关产业：房地产行业、建造行业、金融行业、以及零售行业 -> 尽可能避免红海，努力跨行业寻找蓝海

未来规划:

装修估价 -> 装修承包商管理
定义装修项目(厨房/卧室...) -> 用户发布 -> 承包商报价
衡量标准 -> 完成多少房屋建造 + 质量 
平台专员不定期进行用户走访调查，品质验证，保证数据准确度
当用户面对不同建造商价格，平台只为制定唯一定价标准
当用户选好建造商，双方需安排一致认可的安排表（何时买料？何料优先购买？何时完成相应阶段？... ...）
当双方制定装修安排表，由平台帮助双方安排各自时间（用户何时购买相应材料？建造商何时完成材料安装？... ...）
筛选建造商（承包商），评分系统（牌照+保险+接单历史记录）最最最最最重要，杜绝刷单（比如：一个承包商不可能在同一时间段承包过多工程，视员工人数而定），提升信誉至关重要。
