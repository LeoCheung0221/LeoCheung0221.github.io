数组指针

也称为 行指针，只是一个变量

定义： int (*p)[n]

优先级高，首先说明 p 是一个指针，指向一个整型的一维数组，这个一维数组的长度是n，也可以说是 p 的步长。也就是说执行 p+1 时，p 要跨过 n 个整型数据的长度。

int a[3][4];

int (*p)[4]; // 该语句是定义一个数组指针，指向含4个元素的一维数组。

p = a; // 将该二维数组的首地址赋值给 p，也就是 a[0] 或 &a[0][0]。

p++; // 该语句执行后，也就是 p = p+1；p 跨过行 a[0] 指向了行 a[1][]
